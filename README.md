# Cidadania Digital e Inteligência Artificial: O Impacto das Deepfakes e da Desinformação na Sociedade

## 🎯 Objetivo do Site
Este site foi desenvolvido com o propósito de conscientizar a comunidade escolar (alunos, professores e responsáveis) sobre os riscos associados ao uso indevido de Inteligências Artificiais. O foco principal é ensinar de forma prática como identificar mídias manipuladas (deepfakes), compreender os perigos das fake news automatizadas e promover boas práticas de segurança e cidadania no ambiente digital.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
